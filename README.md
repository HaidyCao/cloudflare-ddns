# CLOUDFLARE-DDNS

更新cloudflare的dns记录

## Params

```
  -N string
        new ip
  -T int
        TTL (default 60)
  -a string
        cloudflare api key
  -i int
        interval (default 300)
  -j string
        url json path
  -n string
        domain name (default "@")
  -p string
        url parse type: full(default) or json (default "full")
  -t string
        domain type (default "A")
  -u string
        default check url (default "http://api.ipify.org")
  -z string
        cloudflare zone id

```
